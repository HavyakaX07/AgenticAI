## 💡 Key Concepts

### System Components
```
User Query → RAG (30ms) → LLM (120ms) → Device Resolver (<1ms) 
  → OPA Policy (15ms) → API Call (85ms) → Response (287ms total)
```

### Tech Stack
```
LLM:        Ollama (llama3.2:3b)
Embeddings: BAAI/bge-small-en-v1.5 (384 dims)
Vector DB:  FAISS IndexFlatIP
Database:   PostgreSQL 15 + In-memory cache
Policy:     Open Policy Agent (OPA)
Framework:  FastAPI + Docker Compose
```

### Performance
```
RAG:                30ms (10% of total time)
Device Resolution:  <1ms (with cache)
Total Request:      287ms average
Throughput:         33 req/sec
```
